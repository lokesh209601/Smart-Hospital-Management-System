# Smart-Hospital-Management-System

Tech Stack: C++ | OOP (Inheritance, Polymorphism, Encapsulation) | Qt Creator | Qt Widgets | File I/O

Designed and developed a full-featured desktop Hospital Management System in C++ using Qt Creator, applying core OOP principles — classes, inheritance, polymorphism, and encapsulation — to model real-world hospital entities such as Patients, Doctors, Appointments, and Billing records.
Built an interactive GUI using Qt Widgets, enabling hospital staff to register patients, schedule appointments, manage doctor availability, and generate bills through an intuitive, form-based interface — eliminating manual record-keeping.
Implemented a modular, layered architecture separating UI logic (Qt forms), business logic (C++ classes), and data persistence (file I/O), ensuring clean, maintainable, and extensible code across the system.
Designed core modules including Patient Registration, Doctor Management, Appointment Scheduling, and Invoice Generation, each as independent, reusable C++ components — demonstrating strong system design and software engineering practices.
Applied object-oriented design patterns to handle polymorphic behavior across different staff roles (e.g., general physician vs. specialist), showcasing the ability to write scalable, real-world software.
